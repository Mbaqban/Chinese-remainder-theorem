# Chinese-remainder-theorem
implementation of Chinese remainder theorem algorithm in python


## Installation

its script dont need installation

## Usage
run script and  inter your equivalent equations
example :
X ≡ 0(mod 3)
X ≡ 3(mod 4)
X ≡ 4(mod 5)
```terminal
Number Of Equivalent Equations : 3
a1 : 0
n1 : 3
a2 : 3
n2 : 4
a3 : 4
n3 : 5

>>> X : 39.0
>>> 39.0 ≡ 0(mod 3) | True
>>> 39.0 ≡ 3(mod 4) | True
>>> 39.0 ≡ 4(mod 5) | True
```
